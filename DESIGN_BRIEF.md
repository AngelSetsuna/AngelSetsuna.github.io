# Angel Setsuna 作品集網站 — 設計簡報（給 claude.ai/design）

> 用法：登入 claude.ai/design，新建一個 Design 專案，把下面整段（或你想改的部分）貼進對話框，再補一句「請幫我設計新版」。也可以把 `works/01.jpg`～`05.jpg` 與 `assets/avatar.png` 一起上傳當參考。

---

## 一句話定位

二次元插畫家 **Angel Setsuna**（奈奈）的個人作品集兼接案網站。深色極簡、以作品圖為主角，需同時服務「商業/法人」與「個人委託」兩種客群。**桌機與手機都要好看。**

## 風格與氛圍

- **深色極簡（dark, minimal）**，沉穩、有質感、不花俏，讓插畫本身最突出。
- 大量留白、細線分隔、克制的動效（淡入、緩慢放大、滑順捲動）。
- 高級感但親和，不要科技冷感、不要商業俗氣。

## 配色（沿用現有主題）

| 用途 | 色碼 |
|------|------|
| 背景（深炭微暖） | `#1b1a1f` |
| 卡片／區塊底 | `#25232b` |
| 分隔線 | `#35323d` |
| 主文字 | `#f0eef3` |
| 次要文字 | `#a6a0b0` |
| 強調色（玫瑰粉） | `#ef7ca3` |
| 強調淡底 | `rgba(239,124,163,.15)` |

圓角小（約 4px）。強調色只用在按鈕、語言切換 active、hover 等重點處。

## 字體

- 日文／內文：**Noto Sans JP**（細字 300 為主）
- 英文／標題／按鈕：**Space Grotesk**（字距略寬，letter-spacing ~.03em）

## 三語需求

介面需支援 **日文(JP) / 英文(EN) / 中文(中)** 切換，右上角放語言切換鈕。預設日文。

## 頁面結構（單頁滾動 + 錨點導覽）

固定頂部導覽列（捲動後加深色半透明 + 毛玻璃）。導覽項目：Works / About / Business / Commission / Contact + 語言切換。

1. **Hero** — 全螢幕，背景為作品輪播幻燈片（緩慢放大淡入），中央放圓形頭貼 + 名字「Angel Setsuna」+ 副標「キャラクターデザイン / ゲームイラスト」+ 社群圖標 + 主要 CTA「お仕事のご依頼」+ 向下捲動指示。
2. **Works（作品牆）** — 響應式網格，點圖開 lightbox（可左右切換）。作品多於 2 種分類時顯示篩選列。附保密聲明（部分作品因 NDA 不公開）。
3. **About** — 自我介紹 + 兩欄：對應分野（角色設計、遊戲立繪/CG、書籍封面、宣傳插畫）與制作環境（CLIP STUDIO PAINT、Photoshop、交付 PSD/PNG/高解析）。
4. **Business（商業/法人）** — 主打商業案件。含：對應分野標籤、5 步合作流程（問い合わせ→見積もり→契約→ラフ制作→納品）、取引條款（授權/買斷、發票、NDA、實績公開可否）、CTA。
5. **Commission（個人委託）** — 4 步流程（表單記入→確認/着手金→ラフ制作→納品），費用說明，下載委託表單按鈕。
6. **Contact** — 雙軌：商業/法人（推薦，mailto 按鈕）與個人（下載表單 + 三步寄送說明）。底部一鍵複製 Email。
7. **Footer** — 社群圖標 + 版權。

## 互動元件

- 三種按鈕樣式：primary（粉底）、outline（描邊）、ghost（淡）。
- 作品 lightbox（關閉 / 上一張 / 下一張）。
- Email 一鍵複製。
- 手機版漢堡選單。

## 雙軌接案的視覺重點

商業/法人是主要收入來源，**Contact 與 Business 區塊要讓「商業委託」更顯眼**（例如標「推薦」標籤、放在前面、用 primary 按鈕），個人委託為輔。

### 可選的延伸需求（要新版更進化時加）

- 作品牆改用 masonry / 不規則排版增加雜誌感。
- Hero 加入更精緻的進場動畫。
- 增加「實績 logo 牆」或「客戶推薦」區塊提升商業信任感。
