# Angel Setsuna — 作品集網站

深色極簡、三語（日／英／中）、純靜態，主打向遊戲公司與出版社接案。
無需後端、無需建置工具，雙擊 `index.html` 即可預覽。

---

## 📁 檔案結構

```
index.html              網站主結構（一般不用動）
assets/
  css/style.css         外觀樣式（想改顏色看最上面 :root）
  js/works.js     ★ 你最常編輯：新增/修改作品
  js/i18n.js      ★ 修改各區塊文字（三語）、社群連結
  js/main.js            程式邏輯（不用動）
works/                  ★ 把作品圖片放這裡
```

★ = 你平常只會碰這三個地方。

---

## ✍️ 最常做的三件事

### 1. 新增一張作品
1. 把圖丟進 `works/`（例如 `07.jpg`）
2. 打開 `assets/js/works.js`，複製一整段 `{ ... }`，改成新作品內容
3. 存檔、重新整理網頁 → 自動排版、自動套三語、自動歸類

### 2. 改文字（標語、自介、流程…）
打開 `assets/js/i18n.js`，找到對應文字改掉。三語各一份，記得三個都改。

### 3. 填社群連結
`assets/js/i18n.js` 最下面 `SOCIAL` 陣列，把網址換成你的帳號。
ArtStation 開好後，把那行的 `//` 註解拿掉並填網址即可。

## 🎞️ 幻燈片：以後想換成橫圖怎麼做

幻燈片預設播 `works.js` 的前幾張。要**精準控制**播哪些、怎麼裁，用兩個可選欄位：

- **只想播特定（橫）圖**：在那些作品加上 `hero: true`。只要有任何一張標了，幻燈片就「只」播有標的那些。
- **橫圖聚焦置中**：加上 `focus: "center"`（直圖預設聚焦上方臉部，不用寫）。

範例（works.js）：
```js
{ image: "works/10.jpg", category: "character",
  title: {...}, client: "Personal", role: {...}, year: 2026,
  hero: true, focus: "center" }   // ← 進幻燈片、置中聚焦
```

## 🖼️ 別人進網站會看到破圖嗎？

**只要遵守一條規則就不會**：圖片放在 `works/`、`assets/`，網站用的是「相對路徑」，所以
**整個 `ask` 資料夾一起上傳**到空間時，圖片會跟著走、不會遺失。

會破圖的情況只有三種，避開即可：
1. 上傳時漏掉 `works/` 或 `assets/` 資料夾 → 要整包一起傳
2. 檔名和 `works.js` 對不上（大小寫也要一致）→ 例如寫 `01.JPG` 但檔案是 `01.jpg`
3. 只把 `index.html` 單獨傳、沒傳圖 → 要整包

> 現在檔案都在你電腦上，只有你看得到。要讓「別人」能進，需先**上線**（見下方）。
> 上線時把整包傳上去，就跟你本機看到的一模一樣，**不會破圖**。先安心放圖就對了。

> 委託信箱已設為 **angel.setsuna.drawing@gmail.com**（與表單一致），要改在 `index.html` 的 `#emailText` 處。

## 📝 委託流程怎麼運作（雙軌）

Contact 區分兩條路：

- **商業・法人合作（主，推薦）**：點「來信洽談」→ 開啟 Email 並自動帶入收件人與主旨。
  不公開固定價，避免被個人小額價錨定，保護商業報價。
- **個人委託（輔）**：點「下載委託表單」→ 下載 `form/` 裡的 .docx → 填好後寄到下方 Gmail。
  **個人價格只寫在表單裡，網站不顯示。**

> 個人委託漲價時：只要把新的 .docx 覆蓋到
> `form/AngelSetsuna_Commission_Form_2026.docx`（檔名不變），**網站完全不用改**。

## 🚀 免費上線（擇一）

- **Cloudflare Pages / Netlify**：把整個資料夾拖進去就好，自動給網址，可綁自己的網域。
- **GitHub Pages**：上傳到 repo，開 Pages 即可。

三者皆免費、皆支援 HTTPS（表單複製 Email 等功能需要 HTTPS）。
